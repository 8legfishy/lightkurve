Systematics correction classes¶

Defines KeplerCBVCorrector, SFFCorrector, and PLDCorrector.

These classes are intended to help remove instrument systematics from time series photometry data.

Classes¶

SFFCorrector(lightcurve) Implements the Self-Flat-Fielding (SFF) systematics removal method.
PLDCorrector(tpf) Implements the Pixel Level Decorrelation (PLD) systematics removal method.
KeplerCBVCorrector(lc_file[, likelihood, prior]) Remove systematic trends from Kepler light curves by fitting Cotrending Basis Vectors (CBVs).
