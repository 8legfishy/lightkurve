API¶

LightCurve objects¶

LightCurve objects contain time-series data on the brightness of a star. They provide easy access to a range of operations, e.g. folding, binning, plotting, and a variety of signal processing. Light curves observed by Kepler or TESS have a specific sub-class which provide extra metadata, but the generic LightCurve object can be used for any data set.

LightCurve([time, flux, flux_err, …]) Implements a simple class for a generic light curve.

TessLightCurve([time, flux, flux_err, …]) Defines a light curve class for NASA’s TESS mission.

Opening data files¶

The lightkurve.search modules makes it easy to open data products from Kepler and TESS, and search for them at the data archive.

search_targetpixelfile(target[, radius, …]) Searches MAST for Target Pixel Files.
search_lightcurvefile(target[, radius, …]) Returns a SearchResult with MAST LightCurveFiles which match the criteria.
open(path_or_url, **kwargs) Opens a Kepler or TESS data product.

LightCurveFile objects represent files that are used to store LightCurves and their metadata. Files of this type are found at NASA’s data archives.

KeplerLightCurveFile(path[, quality_bitmask]) Defines a class for a given light curve FITS file from NASA’s Kepler and K2 missions.
TessLightCurveFile(path[, quality_bitmask]) Defines a class for a given light curve FITS file from NASA’s TESS mission.

TargetPixelFile objects hold the sequence of images (pixels) which can be converted into LightCurve objects using different techniques.

KeplerTargetPixelFile(path[, quality_bitmask]) Defines a TargetPixelFile class for the Kepler/K2 Mission.
TessTargetPixelFile(path[, quality_bitmask]) Defines a TargetPixelFile class for the TESS Mission.
