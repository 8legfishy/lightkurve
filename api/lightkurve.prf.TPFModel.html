TPFModel¶

class lightkurve.prf.TPFModel(star_priors=None, background_prior=<BackgroundPrior( flux=<FixedValuePrior(value=[0])>)>, focus_prior=<FocusPrior( scale_col=<GaussianPrior(mean=[1], var=[0.0001])> scale_row=<GaussianPrior(mean=[1], var=[0.0001])> rotation_angle=<UniformPrior(lb=[-3.1415], ub=[3.1415])>)>, motion_prior=<MotionPrior( shift_col=<GaussianPrior(mean=[0], var=[1.])> shift_row=<GaussianPrior(mean=[0], var=[1.])>)>, prfmodel=None, fit_background=True, fit_focus=False, fit_motion=False)¶

Bases: object

A model which describes a single-cadence Kepler image.

Attributes
star_priorslist of StarPrior objects.

List of stars believed to be in the image.

background_priorBackgroundPrior object.

Beliefs about the background flux.

prfmodelKeplerPRF object.

The callable Pixel Reponse Function (PRF) model to use.

focus_priorFocusPrior object.

Beliefs about the telescope focus.

motion_priorMotionPrior object.

Beliefs about the telescope motion.

fit_backgroundbool

If False, the background parameters will be kept fixed.

fit_focusbool

If False, the telescope focus parameters will be kept fixed.

fit_motionbool

If False, the telescope motion parameters will be kept fixed.
