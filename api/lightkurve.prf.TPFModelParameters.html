to_array()¶

Converts the free parameters held by this class to an array of size (n,), where n is the number of free parameters.

This method exists because scipy.optimize can only optimize arrays of real numbers, yet we like to store in the parameters in human-friendly container classes to make the fitted parameters accessible without confusion.

Returns:
array : array-like

Array containing all the free parameters.
